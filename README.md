# Brush & Base

Paint-night notes — recipes, colours, and photos from the hobby desk.

**Live site:** [tyberium.github.io/brush-and-base](https://tyberium.github.io/brush-and-base/)

## Repo layout

| Path | Purpose |
|------|---------|
| `index.md` | Homepage (GitHub Pages) |
| `entries/` | One markdown file per model or session |
| `assets/photos/` | Diary page photos and mini shots |

## Photo naming

`{entry_slug}_{DDMMYYYY}.jpg` — e.g. `ahriman_11062026.jpg`

Multiple photos from the same session: append `_01`, `_02`, etc. — e.g. `exalted_sorcerer_14062026_01.jpg`

## Paint night workflow

1. Photo the minis and diary page
2. Add or update a file in `entries/`
3. Drop photos in `assets/photos/` using the naming above
4. Commit and push — site rebuilds automatically
